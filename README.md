# 🌸 Mini Pokémon Adventure 🐾

A cozy pixel-art arcade game inspired by **Pac-Man** and the relaxing indie aesthetic of **Sprout Valley**.

In this game, you play as an adorable pixel cat exploring a pastel cottagecore world while collecting cute items, avoiding strawberry-themed enemies, and racing against time.

Built using **Python** and **Pygame** as a final project for **Code in Place**.

---
